Base do Projeto

Projeto Front End, utilizando HTML5, CSS3 e JavaScript.

Sobre o Projeto

O HTML foi utilizado para estruturar o conteúdo da página, o CSS para definir o layout e o estilo visual, e o JavaScript para adicionar interatividade, tornando a calculadora funcional. O objetivo do projeto era criar uma calculadora virtual que simulasse uma calculadora física. Ela possui aparência de uma calculadora real, pode ser ligada e desligada, e cada botão funciona como em um dispositivo físico. Implementei um visor que armazenava cada dígito pressionado como uma String (texto). Ao apertar o botão de resultado, a função eval interpretava a String como uma expressão matemática, permitindo que o cálculo fosse realizado e exibido no visor.

Busca de realismo

Com o JavaScript, criei uma função de ligar e desligar a calculadora, ativando ou desativando todos botões que não sejam o próprio botão Power, também limpando o visor ao ligar e desligar.

Visando UI/UX

O projeto conta com um sistema de temas claro e escuro, que altera cores da calculadora e salva a preferência do usuário para ser mantida ao recarregar a página. Usei o :root no CSS para definir variáveis de cores do tema padrão e criei uma classe .light-theme com cores alternativas para o tema claro. No JavaScript, adicionei uma função que alterna essa classe no <html> ao clicar no botão, permitindo que o usuário troque entre os temas, e salvei a escolha no localStorage para manter o tema ao recarregar a página.
